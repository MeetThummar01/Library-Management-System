# 📚 Library Management System

A console-based **Library Management System** built using **Java** to demonstrate Object-Oriented Programming (OOP) concepts and CRUD operations. This project provides a simple menu-driven interface to manage books efficiently using **ISBN** as a unique identifier.

---

## ✨ Features

- ➕ Add a new book
- ❌ Remove a book
- 🔍 Search a book by ISBN
- 📖 Display all books
- 💰 Update book price
- 📦 Update book stock
- 🚫 Prevent duplicate ISBN entries
- 🖥️ Menu-driven console application

---

## 🛠️ Tech Stack

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (`ArrayList`)
- IntelliJ IDEA / VS Code

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or above
- IntelliJ IDEA / Eclipse / VS Code

### Clone the Repository

```bash
git clone https://github.com/your-username/Library-Management-System.git
```

### Navigate to the Project

```bash
cd Library-Management-System
```

### Compile

```bash
javac *.java
```

### Run

```bash
java Main
```
